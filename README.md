# 在线题库平台(系统)前端

在线考试系统

前后端一起的项目，简易版答题系统，专科毕业设计，从0开始构建  
前端使用了vue3+ts+unocss+vite3等技术最后一次更新依赖在2022年的11月16号
后端使用nestjs+prisma+postgresql

```bash
npm install -g pnpm
pnpm install
pnpm dev

#打包
pnpm build
```
